## 🔧 故障排除

### **如果看不到新功能：**
```bash
# 1. 强制刷新浏览器
Ctrl+F5 (Windows) 或 Cmd+Shift+R (Mac)

# 2. 清除缓存
访问: http://localhost:3004/clear-cache.html
点击"自动清理缓存"

# 3. 检查服务器
确保看到: ➜ Local: http://localhost:3004/
```

### **如果编辑功能异常：**
```bash
# 1. 检查浏览器控制台
F12 → Console → 查看错误信息

# 2. 检查localStorage
F12 → Application → Storage → Local Storage
查看是否有 'userProfile' 数据

# 3. 尝试重新加载页面
```
