interface UserProfile {

  avatar: string;       // 头像 (base64)

## 🔐 安全特性

### **数据验证**
- **必填字段**: 用户名、邮箱不能为空
- **格式验证**: 邮箱格式、密码长度检查
- **一致性检查**: 确认密码必须一致

### **密码安全**
- **最小长度**: 新密码至少6位
- **确认密码**: 双重输入确认
- **当前密码**: 修改时需要验证当前密码

### **数据存储**
- **本地存储**: 使用localStorage临时存储
- **数据清理**: 退出登录时清除敏感数据
- **错误处理**: 异常情况的友好提示
